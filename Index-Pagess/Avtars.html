    Avtar

    Avtar is basiclly What we see as Profile picture in any page like Sign-up, Account page and many more..

    Examples of Avtar

    Here Avtar is Available in 3 different Types Round Avtar, Square Avtar & Round Text avtar with four different size. to use these avtar just initiate your class with 'round-Av' for Round Avtar and 'Sq-Image-class' for square Avtar and use different size of Avtar as per your desire.

    Round Avtars

    For Round Avtar use 'round-Av' and Size of Class (eg. class="round-Av xlg-size-img")

    Square Avtars

    For Square Avtar use 'Sq-Image-class' and Size of Class (eg. class="Sq-Image-class xlg-size-img")

    Text Avtars

    For Text Avtar use 'Primary-Container round-Av' and Size of Class (eg. class="Primary-Container round-Av xlg-size-img")
